# EasyChoose methodology

## Cómo construye la recomendación

EasyChoose prioriza el contexto del usuario antes que la ficha técnica aislada.

1. Identifica la categoría de producto.
2. Recoge el uso real que se le va a dar.
3. Ajusta presupuesto y preferencias.
4. Devuelve recomendaciones que el usuario puede revisar con ayuda de IA.

## Principios

- Uso real antes que especificaciones difíciles de interpretar.
- Valor por precio dentro del presupuesto disponible.
- Recomendaciones explicadas en lenguaje claro.
- Posibilidad de resolver la última duda mediante chat IA.

## Transparencia

EasyChoose puede monetizar mediante enlaces de afiliación sin coste adicional para el usuario final. La recomendación intenta ajustarse a la necesidad del usuario, no a una marca patrocinada.
